Framework Used:
Rest Assured framework is being used with Cucumber to automate API GetQuotes. Spark Extent report is used to generate reports. Maven is used for dependencies.

Feature File:
Fetaure file is places in src/test/resources/features folder.

Step Definition/Hooks:
Step definition is placed in src/test/java.

DataConfig:
Properties file is placed in DataConfig/Config folder.

Extent Report:
Latest Extent report is generated in test-output/SparkReport folder with name "Index.html"

Jenkins Run:
Screenshot from Jenkins build is placed in JenkinsBuild folder
